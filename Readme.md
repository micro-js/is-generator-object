Check if generator.

## Installation

    $ npm install @micro-js/is-generator

## Usage

```js
var isGenerator = require('@micro-js/is-generator')

isGenerator(function * () {
  yield 1
}()) // => true

```

## API

### isGenerator(obj)

- `obj` - object to test

**Returns:** boolean
